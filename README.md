## ✨ Features

- 🎨 **Animated Hero** — Interactive particle network with geometric shapes that react to mouse movement
- 🌙 **Dark Sidebar Navigation** — Fixed sidebar with smooth page transitions
- 👤 **About Section** — Personal intro with scroll-reveal animations
- 🛠️ **Skills Cards** — Visual skill cards with hover effects
- 💼 **Portfolio Gallery** — Filterable project cards with detail pages and image sliders
- 📋 **Resume Timeline** — Work & education history in a clean timeline layout
- 🖊️ **Blog Section** — Article cards with full detail view
- 📬 **Contact Page** — Animated network background with contact form
- 📤 **Share Projects** — Working share buttons (Facebook, Twitter, LinkedIn, WhatsApp, Copy link)
- 📱 **Fully Responsive** — Works on all screen sizes
